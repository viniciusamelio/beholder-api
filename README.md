# beholder api

Backend for Project Beholder which aims to provide an event driven application
monitoring system that offers replay feature out of the box

### Stack

- Web framework - [ElysiaJS](https://elysiajs.com/)
- Linter - [Biome](https://biomejs.dev/)
- ORM - [Drizzle](https://orm.drizzle.team/) ([SQLite](https://sqlite.org/))
